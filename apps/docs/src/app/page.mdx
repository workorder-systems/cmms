## Getting started {{ anchor: false }}

You need your Supabase project URL and anon key from the [Supabase dashboard](https://supabase.com/dashboard) (Project Settings → API). Install the SDK, create a client, then sign in with Supabase Auth. For tenant-scoped data, set tenant context and refresh your session. {{ className: 'lead' }}
