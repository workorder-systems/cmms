# API Documentation

This API is **Supabase-backed**: reads go through public views, writes through RPCs. The **`@workorder-systems/db`** package wraps that surface with TypeScript types for tenants, work orders, assets, labor, reporting, and the rest of the domain. {{ className: 'lead' }}

## Getting started {{ anchor: false }}

1. Copy your **project URL** and **anon key** from the [dashboard](https://supabase.com/dashboard) → **Project Settings → API**.
2. Install **`@workorder-systems/db`** and **`@supabase/supabase-js`**, then call **`createDbClient(url, anonKey)`**.
3. Sign users in with **Supabase Auth**. For tenant-scoped data, set tenant context and refresh the session so the JWT includes **`tenant_id`**.
