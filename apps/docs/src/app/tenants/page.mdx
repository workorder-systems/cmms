# Tenants

Lists tenants the current user is a member of; create tenants, invite users, assign roles. Tenant context is **not** required for tenants; use `setTenant` for work orders, assets, and other tenant-scoped resources. {{ className: 'lead' }}

## List tenants

List all tenants the authenticated user is a member of (from `v_tenants`).

<CodeGroup title="client.tenants.list()">

```ts
const tenants = await client.tenants.list()
// Returns TenantRow[]
```

</CodeGroup>

## Get by ID

Fetch a single tenant by ID.

<CodeGroup title="client.tenants.getById(id)">

```ts
const tenant = await client.tenants.getById('uuid-of-tenant')
// Returns TenantRow | null
```

</CodeGroup>

## Create tenant

Create a new tenant. The authenticated user becomes a member and is assigned the admin role. Returns the new tenant UUID.

<CodeGroup title="client.tenants.create(params)">

```ts
const tenantId = await client.tenants.create({
  name: 'Acme Corp',
  slug: 'acme-corp',
})
// Returns string (UUID)
```

</CodeGroup>

<Properties>
  <Property name="name">
    Required. Tenant display name.
  </Property>
  <Property name="slug">
    Required. Unique slug.
  </Property>
</Properties>

## Invite user

Invite a user to a tenant with a role (e.g. `admin`, `member`). The invitee receives an invite; they must accept to join.

<CodeGroup title="client.tenants.inviteUser(params)">

```ts
await client.tenants.inviteUser({
  tenantId: 'uuid-of-tenant',
  inviteeEmail: 'newuser@example.com',
  roleKey: 'member',
})
```

</CodeGroup>

<Properties>
  <Property name="tenantId">
    Required. Tenant UUID.
  </Property>
  <Property name="inviteeEmail">
    Required. Invitee email.
  </Property>
  <Property name="roleKey">
    Required. Role key (e.g. `admin`, `member`).
  </Property>
</Properties>

## Assign role

Assign a role to an existing user in a tenant.

<CodeGroup title="client.tenants.assignRole(params)">

```ts
await client.tenants.assignRole({
  tenantId: 'uuid-of-tenant',
  userId: 'uuid-of-user',
  roleKey: 'member',
})
```

</CodeGroup>

<Properties>
  <Property name="tenantId">
    Required. Tenant UUID.
  </Property>
  <Property name="userId">
    Required. User UUID (Supabase Auth).
  </Property>
  <Property name="roleKey">
    Required. Role key.
  </Property>
</Properties>
